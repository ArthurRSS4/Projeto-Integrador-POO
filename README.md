# Projeto Integrador – Sistema de Gestão de Assinaturas de Streaming
Este projeto foi desenvolvido como parte de um trabalho acadêmico de Programação Orientada a Objetos em Python. O sistema simula a gestão de assinaturas de serviços de streaming, incluindo:

• Cadastro e gerenciamento de usuários

• Criação e controle de assinaturas (ativar, pausar, cancelar, trocar plano)

• Modelagem de planos com preços e características

• Registro de pagamentos

• Persistência de dados em arquivos JSON

• Interface simples em linha de comando (CLI)

A arquitetura segue boas práticas de POO, utilizando herança, polimorfismo, composição, agregação e associação, além de tratamento de exceções e organização modular em pacotes.
